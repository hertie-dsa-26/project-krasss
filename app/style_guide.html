Flask template rules

Every page: {% extends "base.html" %} at the top

Page title: {% block title %}Page Name — Climate & Health Explorer{% endblock %}

Content: wrapped in {% block content %} ... {% endblock %}

Wrapper div: <div class="eda-wrap"> with max-width 1100px centered

D3.js: load from https://cdnjs.cloudflare.com/ajax/libs/d3/7.8.5/d3.min.js

API calls: always use fetch('/api/endpoint') — never hardcode data in the HTML

New routes: add to app/main.py above the if __name__ == '__main__': line
