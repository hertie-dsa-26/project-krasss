About the project

From raw data
to interpretable predictions.

This project integrates public data from NOAA, the CDC, and the U.S. Census to examine how weather patterns — temperature, precipitation, and extreme events — relate to health outcomes like asthma, sleep quality, and mental distress across U.S. metropolitan areas. A custom Kernel Ridge Regression model with a Gaussian kernel was implemented from scratch, without relying on external machine learning libraries.

01
Collection

NOAA, CDC Places, U.S. Census aligned at MSA level

02
Cleaning

Standardised, merged, and transformed into 52 variables

03
EDA

Univariate and bivariate analysis of key patterns

04
Modelling

Custom KRR with Gaussian kernel, fitted per outcome

05
Application

Interactive Flask app with scenario-based forecasting
